# Ecuacion de Segundo Grado
Aplicación web para resolver ecuaciones de segundo grado, desarrollada como proyecto de aprendizaje con HTML, CSS, JavaScript y Python (Flask). El proyecto integra frontend y backend mediante comunicación cliente-servidor para procesar los cálculos y mostrar los resultados.
Para hacer funcionar el código debe:
      1. Descargar todos los archivos adjuntos.
      2. Abrir el html en su navegador.
      3. Ejecutar el servidor en Flask desde VS Code.
      4. ¡Probar la página web!
